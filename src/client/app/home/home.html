What it's about

This is a simple application built using Node.js, Express, Karma, Jasmine, Gulp, AngularJS and Less.js with a focus on best practices in both web development and web design.

The approach

This is an open source project that closely follows the Angular Style Guide recommended by AngularJS. Next step is to incorporate E2E tests.
