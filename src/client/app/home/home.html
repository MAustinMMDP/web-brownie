What it's about

This is a simple code base using Node.js, Express, Gulp, AngularJS and Less.js with a focus on best practices in both web development and web design.

The approach

This code base closely follows the Angular Style Guide recommended by AngularJS. Next steps for this build is to incorporate unit testing.
